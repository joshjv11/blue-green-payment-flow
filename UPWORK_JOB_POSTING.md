## Job Description

We're looking for an experienced full-stack developer to work on **InvoiceFlow**, a production-ready SaaS platform for business management and GST compliance in India.

### About the Project

InvoiceFlow is a comprehensive business management platform that combines:
- Invoice/bill management with automated reminders
- Sales and purchase order management
- Inventory tracking and analytics
- Complete GST compliance automation (E-Invoicing, GSTR Filing, ITC Reconciliation)
- Financial reporting (P&L, Balance Sheet, Cash Flow)
- WhatsApp integration for customer communication
- OCR invoice processing
- AI-powered financial insights

**Current Status:** Live application with active users, ongoing enhancements

### Technology Stack

**Frontend:**
- React 18.3 with TypeScript
- Vite, Tailwind CSS, shadcn/ui
- TanStack Query, React Router
- Recharts for data visualization

**Backend:**
- Supabase (PostgreSQL, Edge Functions, Auth, Storage)
- Node.js/Express for additional APIs
- Deno for Edge Functions
- BullMQ + Redis for queues

**Integrations:**
- Razorpay (payment processing)
- Twilio (WhatsApp messaging)
- GSTN API (GST compliance)
- AI APIs (Groq, Lovable Gateway)

**Infrastructure:**
- Vercel (frontend deployment)
- Supabase (backend)
- Comprehensive test suite
