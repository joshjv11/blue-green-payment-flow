## Job Post Title
**Full-Stack Developer Needed - InvoiceFlow SaaS Platform (React + TypeScript + Supabase)**

## Job Description

### About the Project

InvoiceFlow is a **production-ready SaaS platform** for business management and GST compliance, built for Indian businesses. The application combines bill management, expense tracking, GST filing automation, inventory management, and financial reporting in a single platform.

**Current Status:** ✅ Already in production with paying customers  
**Tech Stack:** React 18 + TypeScript, Supabase (PostgreSQL), Deno Edge Functions, Razorpay, Twilio WhatsApp API

---

### What We Need

We're looking for an experienced **full-stack developer** to:

1. **Fix critical feature gating bugs** (HIGH PRIORITY)
   - Free users accessing paid features (revenue loss risk)
   - Need to implement proper plan validation and expiration checks
   - Add database-level RLS policies

2. **Implement pricing restructure** (HIGH PRIORITY)
   - Migrate from 3-tier to 4-tier pricing model
   - Update payment configuration, database schema, and UI

3. **Test and polish existing features**
   - End-to-end testing of payment flows, GST features
   - Improve error handling and code quality

4. **Ongoing development and maintenance**
   - Add new features as prioritized
   - Bug fixes and performance optimizations

---

### Required Skills

**Must Have:**
- ✅ 3+ years React + TypeScript experience
- ✅ Experience with Supabase (or Firebase/AWS Amplify)
- ✅ PostgreSQL database knowledge
- ✅ Payment integration experience (Razorpay/Stripe)
- ✅ Understanding of SaaS architecture and feature gating

**Nice to Have:**
- ✅ Indian market knowledge (GST compliance, Razorpay)
- ✅ WhatsApp API integration experience
- ✅ OCR/AI integration experience
- ✅ Testing experience (Jest/Vitest)
- ✅ DevOps experience (Vercel, Supabase)

---

### Project Details

**Codebase:**
- 100% TypeScript, well-organized
- 50+ database tables, optimized with indexes
- 40+ Edge Functions for business logic
- Comprehensive documentation available

**Key Features (28+ already implemented):**
- Bill/invoice management
- Payment processing (Razorpay)
- GST compliance (E-Invoicing, GSTR Filing, ITC Reconciliation)
- WhatsApp integration
- OCR invoice processing
- Financial reports (P&L, Balance Sheet, Cash Flow)
- Advanced analytics dashboard
- Admin panel

**Recent Improvements:**
- Dashboard performance optimized (3-7s → <1s load time)
- WhatsApp broadcast queue system (handles 10,000+ messages)
- Payment auto-verification implemented
- ITC reconciliation now works with real GSTN API
