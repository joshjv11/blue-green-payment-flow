# InvoiceFlow - Project Description (Short Version)

## Quick Overview

**InvoiceFlow** is a production-ready SaaS platform for business management and GST compliance in India. The application combines invoice management, inventory tracking, sales/purchase orders, financial reporting, and complete GST automation.

**Status:** Live application with active users  
**Tech Stack:** React/TypeScript, Supabase, PostgreSQL, Node.js, Razorpay, Twilio

---

## Technology Stack

**Frontend:** React 18 + TypeScript, Vite, Tailwind CSS, shadcn/ui, TanStack Query  
**Backend:** Supabase (PostgreSQL, Edge Functions), Node.js/Express, Deno  
**Integrations:** Razorpay, Twilio (WhatsApp), GSTN API, AI APIs (Groq, Lovable)  
**Infrastructure:** Vercel (frontend), Supabase (backend), Redis (queues)

---

## Key Features

- **Invoice/Bill Management** - Full lifecycle management with reminders
- **Sales & Purchase Orders** - Complete order management system
- **Inventory Management** - Stock tracking, movements, analytics
- **GST Compliance** - E-Invoicing, GSTR Filing, ITC Reconciliation, automated compliance
- **Financial Reports** - P&L, Balance Sheet, Cash Flow statements
- **WhatsApp Integration** - Automated reminders and messaging
- **OCR Processing** - Automatic invoice data extraction
- **AI Features** - Financial coaching, spending insights
- **Analytics Dashboard** - Real-time business metrics and KPIs
